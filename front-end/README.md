# Front-end (React + Vite)

Run locally:

```bash
cd front-end
npm install
npm run dev
```

The dev server will run on http://localhost:5173 and talk to the backend at http://localhost:3000/api (make sure the backend is running and CORS is enabled).

Features:
- List users
- Create role
- Create user (specify role by name)

You can extend components for memberships, contributions, and payments similarly.
